# Code of Conduct

OpenOyster follows a simple standard: be respectful, precise, and constructive.

Contributors are expected to:

- Debate ideas, not people.
- Explain trade-offs clearly.
- Assume good faith while still asking for evidence.
- Avoid harassment, personal attacks, discriminatory language, or intimidation.
- Prioritise safety and traceability in autonomous-system design.

Maintainers may moderate issues, discussions, or pull requests that violate these expectations.
